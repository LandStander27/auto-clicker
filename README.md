## Auto clicker
- An auto clicker made in Rust, not that complicated
- Platforms:
	- **Windows**: Works 100%
	- **Linux/MacOS**: Should work, however it has not been tested

***

#### Building
1. Clone the repo
2. `cargo b --release`
3. Executable will be in `./target/release`
